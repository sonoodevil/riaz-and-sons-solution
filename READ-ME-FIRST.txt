RIAZ & SONS SOLUTION — READY-TO-UPLOAD WEBSITE (public_html)
============================================================
This is the LIGHT version (2.5 MB, no heavy video files) made for shared
.com.pk hosting. Videos use YouTube Unlisted embeds.

WHAT'S INSIDE
  index.html, styles.css, app.js, assets/  (15 real photos)
  -> This is your whole website. No backend needed for the public site.

HOW TO PUT IT ONLINE (cPanel — most .com.pk hosts)
  1. Log in to your hosting cPanel.
  2. Open File Manager -> public_html.
  3. Upload index.html, styles.css, app.js, and the assets/ folder
     (or upload this whole folder's contents). index.html must sit
     directly inside public_html.
  4. Visit https://yourdomain.com.pk  -> the site is live.
  5. cPanel -> SSL/TLS Status -> Run AutoSSL  (free https padlock).

ADD YOUR VIDEOS (YouTube Unlisted — recommended for public)
  Right now each video card shows a clean "Add YouTube Unlisted ID"
  placeholder. To turn them into real videos:
  1. Upload each clip to YouTube -> Visibility: UNLISTED.
  2. Copy the ID from the link  https://youtu.be/THIS_ID
  3. Open app.js, find the VIDEOS list near the top, and paste each ID:
        youtube: "THIS_ID"
  4. For the "Watch Walkthrough" button, set WALKTHROUGH.youtube the same way.
  5. Re-upload app.js. Done — the cards become private YouTube players.

  (The original MP4 files are in the full project zip,
   Riaz-and-Sons-Solution-WEBSITE.zip, under backend/static/assets/videos/.)

OPTIONAL — LIVE BOOKING / 3D TELEMETRY
  Those features need the Python backend (see DEPLOYMENT.md in the full
  project). If you don't host it, the site still works; only the live
  booking-save / telemetry are inactive.
